#  Core Vision

> Một thế giới nơi:
>
> * Không có run nào giống nhau
> * Dungeon luôn thay đổi
> * Player phải thích nghi liên tục
> * Item và skill thay đổi meta mỗi lần chơi
> * Boss không chỉ mạnh mà còn unpredictible
> * Cái chết là một phần progression

```txt
You lose the run.
But never the experience.
```
